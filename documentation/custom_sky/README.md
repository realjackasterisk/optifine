
<div align="center">
    <h1>Custom Sky</h1>
</div>

<div align="center">
    <h2>Description</h2>
</div>

Custom Sky allows you to change the texture of Minecraft's overworld skybox.

<div align="center">
    <h2>In-Game Options</h2>
</div>

```
Video Settings... -> Quality... -> Custom Sky: ON|OFF
```

| Option | Default |
| :---: | :---: |
| `ON` | ✔️ |
| `OFF` |  |
